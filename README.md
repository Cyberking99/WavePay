# WavePay

WavePay is a seamless crypto payment solution designed to make sending and receiving digital assets easy and secure.

## Features

-   **Secure Authentication**: Powered by Privy for seamless and secure user logins.
-   **User Dashboard**: A comprehensive overview of your assets and activities.
-   **Send Crypto**: Easily send digital assets to other users or addresses.
-   **Payment Links**: Create and manage payment links for easy requesting of funds.
-   **Transaction History**: View detailed history of all your transactions.
-   **Profile Management**: Manage your user profile and settings.
-   **Onboarding Flow**: Smooth onboarding experience for new users.

## Tech Stack

This project is built with a modern tech stack ensuring performance and scalability:

-   **Frontend**: [React](https://react.dev/), [Vite](https://vitejs.dev/), [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/), [Shadcn UI](https://ui.shadcn.com/)
-   **State Management**: [TanStack Query](https://tanstack.com/query/latest)
-   **Web3 & Auth**: [Privy](https://www.privy.io/), [Wagmi](https://wagmi.sh/)

## Getting Started

To get started with the project locally, follow these steps:

### Prerequisites

-   Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

1.  Clone the repository:
    ```sh
    git clone https://github.com/Cyberking99/WavePay.git
    ```

2.  Navigate to the project directory:
    ```sh
    cd WavePay
    ```

3.  Install dependencies:
    ```sh
    npm install
    ```

4.  Start the development server:
    ```sh
    npm run dev
    ```

## Project Structure

The project structure is organized as follows:

-   `src/components`: Reusable UI components.
-   `src/pages`: Application pages (Dashboard, Send, etc.).
-   `src/hooks`: Custom React hooks.
-   `src/lib`: Utility functions and configurations.
